# 媒体选择

# 使用说明

<template>
    <sel-media ref="自己定义" v-model="ids"></sel-media>
</template>
<script>
    import selMedia from '@/components/otherDialog/selMedia'
</script>

# 属性

参数 说明 类型 可选值 默认值
value 配合v-model使用得到的是用逗号拼好的媒体id字符串 String _ ""
btnType 按钮类型 String primary / success / warning / danger / info / text text
placement 弹窗出现位置 String top / top-start / top-end / bottom / bottom-start / bottom-end / left/left-start / left-end / right / right-start / right-end left
trigger 触发弹窗事件 String click/focus/hover/manual click
isMapBtn 是否在地图组件上使用，为true时按钮显示的位置发生改变 Boolean _ false
isShowHotMedia 是否显示热门媒体快速选择项 Boolean _ false
btnTxt 按钮文字 String _ 添加
icon 按钮icon String _ el-icon-plus

# 事件

事件名称 说明 回调参数
change 选择值发生变化时会触发 返回所选媒体json数组

# 方法

方法名 说明 参数
returnShow 表单数据回显使用 接收需要回显的媒体id, 必须是数组
