# 成员选择

# 使用说明

<template>
    <sel-user ref="自己定义" v-model="id"></sel-user>
</template>
<script>
    import selUser from '@/components/otherDialog/selUser'
</script>

# 属性

参数 说明 类型 可选值 默认值
value 配合v-model使用， 单选时的是选择的员工id， 多选时得到的是用逗号拼好的员工id字符串 String _ ""
buttonType 按钮类型 String default default
isMoreSel 是否可以选择多个员 Boolean — false
required 员工是否必需选择才能点击确定按钮 Boolean — false
showLeaver 是否显示离职员工 Boolean — false
isCheckbox 是否显示多选复选框 Boolean — false
isBackFun 点击确定按钮是否要触发回调，配合事件backFun 使用 Boolean — false
rejectUsers 控制某些员工不能选择 Array — [] 员工id数组
title 弹窗标题 String — 人员选择

# 事件

事件名称 说明 回调参数
backFun 点击确定后触发回调方法 单选时返回选择的员工的对象，多选时返回选择的员工json数组
change 选择员工后触发 单选时返回选择的员工的id，多选时返回选择的员工id数组

# 方法

方法名 说明 参数
returnUser 单选-表单数据回显方法 (id, label) id是员工id，label是员工名字
returnMoreUser 多选-表单数据回显方法 json数组 参数格式 [{id:'xxxxxxxx', label:'xxxxxxxx'}]
