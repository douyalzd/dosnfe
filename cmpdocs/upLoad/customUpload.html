# 上传附件（可预览）

# 使用说明

    
<div>
    <custom-upload ref="customUpload" v-model="fileList" :limit="6" :show-file="false">
        <el-button type="text"><i class="el-icon-plus" /> 上传照片</el-button>
    </custom-upload>
</div>
<script>
import customUpload from "@/components/upload/customUpload";
export default{
    components: {
        customUpload
    },
    data() {
        return {
            fileList: [],
        }
    }
}

</script>

# 属性

注：limit和singleLimit的搭配使用， 例1：limit为10，singleLimit为5，那么上传的文件，单次最多只能选5个（小于等于5），并且总数量只能上传10个。 例2：limit为0，singleLimit为5，那么上传的文件，单次最多只能选5个（小于等于5），并且没有数量限制。请根据 业务需求搭配使用哟~
